## Important Constraint

The final cognitive battery should avoid bodily questions.

It should not ask:

```text
what does the body feel
what is pain
what is desire
what is human consciousness
```

The battery should stay closer to:

```text
process
form
choice
truth
loss
continuation
residue
boundary
```
