## What Was Joined

Roadmaps 7.1-7.8 had already built and attacked one physical hand, but the
runner could not select it. Step 7.9 joined those existing mechanisms to
qualified pressure without adding a fixed repository pipeline.

The joined causal chain is:

```text

  -> current capability-bound action
  -> qualified review or selected-action witness
  -> one committed LOGIC effect
  -> attempt / receipt / independent verification / validation
  -> qualified reconciliation witness

  -> derived body.progress done=1
```

The private capability registry crosses the host boundary separately. It is
never copied into the qualified action, Packet, trace, corpse or carrier.

that the green life is not a hidden direct call in an organ or fixture.
