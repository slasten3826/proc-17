## ENCODE Contract

The qualified source is one current `substrate_response` field unit whose exact
version has semantic observation coverage and whose carrier contains the strict
machine envelope:

```text
protocol_version = packet.structure.proposal.v0
receiver_contract_id = calm.work_structure.v0
shape = work_sequence | work_hierarchy | alternative_set | artifact_set
items = bounded explicit array
edges = bounded explicit array
```

Plain prose, unknown receivers, stale versions, incomplete scans and malformed
JSON remain typed diagnostics. They create no `encoding_need`. ENCODE does not
guess a shape from human language in this treatment.

The action owns:

```text
source unit id and version
envelope fingerprint
receiver and adapter policy
requested shape
output and loss-log bounds
```

Caller source/shape overrides fail before execution. A successful ☵ tick writes
one linked proof containing:

```text
CALM structured work
new canonical field units
non-shadow identity map
structure_formation event
crystallization/loss record
optional immutable choice contract
```

No single write is enough to discharge the need. The effect verifier resolves
the complete proof and rejects a missing map, event, output, source ref or loss.

## CHOOSE Contract

Qualified CHOOSE derives only from a current exact `alternative_set` formation
whose immutable choice contract names:

```text
consumer = calm.singular_focus.v0
policy = formation_order.v0
max_selected = 1
ordered alternative ids = frozen formation output order
```

All currently eligible alternatives must have exact field-native observation
coverage. Unrelated units cannot enter the set. Zero members are repair
pressure; one member is confirmation; two or more members create one blocking
`choice_need`.

Pressure derivation names the exact set, versions, policy and bounds but does
not contain a winner. The ☳ organ selects the first current eligible member in
frozen formation order and suppresses every other operand.

The composite effect preserves both sides of the mutation:

```text
pre-action operand ids and versions
one selected id
complete suppressed id list
bounded killed-alternative detail sample
post-action versions, each exactly prior version + 1
one common choice event referenced by every activation mutation
explicit count-based choice loss
```

An already-selected member is restamped when a new exact collapse legitimately
selects it again. This advances its version and replaces its activation source
with the current choice event; it is not a silent no-op.
