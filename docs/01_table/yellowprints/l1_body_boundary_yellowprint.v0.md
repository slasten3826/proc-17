## 0. Purpose

This table defines the minimum contract for recovering the omitted live L1
CHAOS field without disturbing the current proc-17 body.

It separates three claims:

```text
P1  the old canonical L1(C) physics can be ported exactly
P2  the port produces non-trivial bounded temporal measurements
P3  those measurements are causally useful to proc-17 routing
```

The first experiment may prove P1. It cannot silently claim P2 or P3.
