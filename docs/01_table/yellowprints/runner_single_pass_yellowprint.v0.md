# Single-Pass Runner Yellowprint

Purpose:

```text
give the rebuilt body a first executable operator route
```

Route:

```text
▽ packet.new
☴ observe
☵ encode
☳ choose
☲ cycle
△ assemble turn manifest
```

The runner is not the final agent loop.

It is the first body-level proof that the new packet can move through organs
without the old CLI carrying the route by hand.

## Boundary

`☴` may call a substrate.

`☵` must read only packet chaos.

`☳` must read calm field/work units and record collapse.

`☲` must decide whether continuation is needed.

`△` may assemble the current turn output.

`△` must not kill the packet while `☲` says `again`.

## Expected First Behavior

For a fake substrate response:

```text
packet is born
observe appends semantic chaos
encode crystallizes chaos into calm work units
choose records selected and killed alternatives
cycle sees remaining pending work
manifest assembles turn-level payload
packet remains running
```

Completion and death belong to a later multi-turn runner, not to this smoke
route.
